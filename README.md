# LinkedIn Job Assistant

AI-powered LinkedIn assistant helping job seekers create personalised messages, connection requests, and InMail to stand out.

Uses your own API key (OpenAI, Google Gemini, or Anthropic) — processed locally in your browser.

## Chrome Web Store

Install from the Chrome Web Store or load unpacked from this repo for development.

## Development

1. Clone this repo
2. Open `chrome://extensions` → Developer mode → **Load unpacked** → select this folder
3. Configure provider, model, and API key in Options
4. Use on LinkedIn profiles, search results, or messaging

### Tests

```bash
npm install
npm test
```
