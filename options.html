How to get a key and typical cost

Pricing changes over time. Figures below are ballpark for light use. Check each vendor’s pricing page before billing.

OpenAI
  1. Open API keys and sign in.
  2. Create a new secret key (starts with sk-).
  3. Optional: set a billing limit.

Rough cost: GPT-4o mini is very cheap for many short generations. See OpenAI pricing.

Google Gemini
  1. Open Google AI Studio and create a key (AIza…).
  2. Review limits and billing.

Rough cost: Flash models are often low cost or have free tier allowances. See Gemini API pricing.

Anthropic (Claude)
  1. Open Anthropic Console and create an API key.
  2. Add billing in the console if required.

Rough cost: Haiku is the budget Claude tier. See Anthropic API pricing.
